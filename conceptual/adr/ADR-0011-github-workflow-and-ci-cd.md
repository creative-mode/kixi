# ADR 0011: GitHub Workflow and CI/CD

## Status
Accepted

## Context

The project requires traceability, code quality, and collaborative development.

---

## Decision

The workflow is:
- Issues tracked in GitHub Projects
- One branch per issue
- Pull Requests with mandatory code review
- GitHub Actions for CI/CD

---

## Consequences

- High code quality
- Clear traceability
- Consistent delivery process

---

## Decision Summary

All development in Kixi follows an issue-driven GitHub workflow with automated CI/CD.
